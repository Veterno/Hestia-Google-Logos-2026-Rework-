## 🚀 Installation Guide

You can use either of the two convenient methods below to apply this CSS style to the Google website in your browser.

### Method 1. One-Click Automatic Installation (Recommended)

The fastest method, which allows the style to receive automatic updates in the future.

1. Open your browser's extension store (Chrome, Edge, Opera, Brave, etc.).
2. Search for and install the **Stylus** extension (recommended — lightweight, ad-free, and open-source) or **Stylish**.
3. Visit the official publication page: [userstyles.org/styles/283892](https://userstyles.org).
4. Click the **"Install Style"** button.
5. Confirm the installation in the extension's pop-up window.

---

### Method 2. Manual Installation via CSS Code

Use this method if you want to modify the source code yourself or install it locally.

#### Step 1. Install a userstyles extension
1. Open your browser's extension store.
2. Install the **Stylus** or **Stylish** extension.

#### Step 2. Create a new style
1. Click the **Stylus** extension icon on your browser toolbar.
2. Click **"Manage"** or **"Write style for"**.
3. If you clicked "Write style for", select **"google.com"** (or enter the URL manually so the style only runs on Google pages).

#### Step 3. Add the CSS code
1. Copy the entire code from the `.css` file in your repository.
2. Paste the copied code into the large text field of the extension editor.
3. In the left column of the editor, enter any name for the style (e.g., `Hestia Google Logos 2026`).
4. Click the **"Save"** button in the top-left corner.

#### Step 4. Verification
Open or refresh the main page of [google.com](https://google.com). The changes and new logos should apply instantly!
