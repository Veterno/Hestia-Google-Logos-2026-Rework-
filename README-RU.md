## 🚀 Как запустить и установить этот CSS-стиль на компьютер

Чтобы применить этот стиль к сайту Google в вашем браузере, выполните следующие шаги:

### Шаг 1. Установите расширение для пользовательских стилей
Для работы CSS-файла вам понадобится специальное расширение, которое умеет внедрять кастомные стили на веб-страницы.
1. Откройте интернет-магазин вашего браузера (Chrome, Edge, Opera, Яндекс.Браузер).
2. Найдите и установите одно из популярных расширений:
   * **Stylus** (рекомендуется — легкое, без рекламы и открытый исходный код).
   * Или **Stylish**.

### Шаг 2. Создайте новый стиль
1. Нажмите на иконку установленного расширения (**Stylus**) на панели инструментов браузера.
2. Нажмите на кнопку **«Управление»** (Manage) или **«Создать стиль»** (Write style for).
3. Если вы нажали «Создать стиль», выберите вариант **«для google.com»** (или введите адрес вручную в поле привязки, чтобы стиль работал только на страницах Google).

### Шаг 3. Добавьте CSS-код
1. Скопируйте весь код из файла `.css` в вашем репозитории.
2. Вставьте скопированный код в большое текстовое поле редактора в расширении.
3. В левой колонке расширения введите любое понятное название для стиля (например, `Hestia Google Logos 2026`).
4. Нажмите кнопку **«Сохранить»** (Save) в левом верхнем углу.

### Шаг 4. Проверка
Откройте или обновите главную страницу [google.com](https://google.com). Изменения и новые логотипы должны сразу примениться!
